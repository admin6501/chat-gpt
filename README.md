## 📋 درباره پروژه

یک ربات فروشگاهی حرفه‌ای برای تلگرام که با Python و کتابخانه [python-telegram-bot](https://python-telegram-bot.org) نوشته شده است. این ربات امکان فروش خودکار اکانت‌های ChatGPT یا هر محصول دیجیتال دیگری را فراهم می‌کند.

## 🚀 ویژگی‌ها

| قابلیت | توضیح |
|:--|:--|
| 👑 پنل ادمین گرافیکی | دسترسی به پنل ادمین با یک دکمه (بدون نیاز به دستور) |
| 👋 متن استارت سفارشی | قابلیت تنظیم پیام خوش‌آمدگویی |

| 📦 مدیریت سفارش‌ها | پیگیری، تأیید و ارسال اکانت‌ها |
| ⏰ لغو خودکار سفارش | لغو سفارش‌های بدون پرداخت با زمان قابل تنظیم |
| ⚙️ تنظیمات کامل | همه چیز از داخل ربات قابل تغییر است |
| 🐳 اجرا با Docker | نصب و مدیریت آسان |
| 💾 بکاپ و بازیابی | پشتیبان‌گیری کامل از دیتا |
| 🗑️ حذف کامل | امکان حذف کامل ربات با یک گزینه |

## 👑 پنل ادمین

وقتی ادمین ربات رو استارت کنه، دکمه **👑 پنل مدیریت** در منوی اصلی نمایش داده میشه.

### امکانات پنل ادمین:

```
📋 سفارش‌های در انتظار    ✅ تایید پرداخت
📤 ارسال اکانت            ⚙️ تنظیمات فروشگاه
```

---

## ⚙️ تنظیمات فروشگاه

تمام تنظیمات از داخل ربات قابل تغییر است:

| تنظیم | توضیح |
|:--|:--|
| 👋 متن استارت | پیام خوش‌آمدگویی |
| 🛒 نام محصول | نام محصول برای نمایش |
| 💰 قیمت محصول | قیمت به تومان |
| 💳 شماره کارت | شماره کارت برای پرداخت |
| ℹ️ درباره محصول | توضیحات محصول |
| 📜 قوانین | قوانین فروشگاه |
| 📞 پشتیبانی | اطلاعات پشتیبانی |
| ⏰ زمان لغو سفارش | زمان لغو خودکار (دقیقه) |
| 🔄 بازه چک سفارش‌ها | فاصله چک کردن سفارش‌ها (ثانیه) |
| 🎟️ مدیریت کدهای تخفیف | افزودن، حذف و مشاهده کدها |

## 🎟️ سیستم کد تخفیف

### افزودن کد تخفیف (مرحله به مرحله):

```
مرحله 1: 📝 کد تخفیف (مثال: SALE20)
مرحله 2: 📊 نوع تخفیف (1=درصدی، 2=مبلغی)
مرحله 3: 💯 مقدار تخفیف
مرحله 4: 📊 حداکثر استفاده کلی (0=نامحدود)
مرحله 5: 👤 حداکثر استفاده هر کاربر (0=نامحدود)
مرحله 6: 📅 تعداد روز تا انقضا (0=بدون انقضا)
```

### امکانات مدیریت کد تخفیف:

- ➕ افزودن کد تخفیف
- 📋 لیست کدها
- 📊 آمار استفاده
- ❌ غیرفعال کردن
- 🗑️ حذف کد

## 🧩 ساختار پروژه

```
chatgpt-seller-bot/
├── bot.py              ← کد اصلی ربات
├── Dockerfile          ← تنظیمات Docker
├── docker-compose.yml  ← تنظیمات Docker Compose
├── .env                ← متغیرهای محیطی
└── data/
    ├── config.json     ← تنظیمات فروشگاه
    └── orders.db       ← دیتابیس سفارش‌ها
```

## 💾 بکاپ و بازیابی

بکاپ شامل:
- `config.json` - تنظیمات

- `.env` - توکن و آیدی ادمین

برای بکاپ: گزینه 6  
برای بازیابی: گزینه 7

## 🧠 نکات فنی

- 🐳 اجرا روی Docker با restart خودکار
- 💾 داده‌ها در پوشه `data/` ذخیره می‌شوند
